# 星星银行

一款面向儿童的积分激励小应用。孩子完成日常任务可以赚取星星积分，攒够积分后可以兑换奖励。界面采用暖色卡通风格，操作简单，适合家长和孩子一起使用。

## 功能概览

底部悬浮导航栏可在四个页面间切换：**任务**、**奖励**、**记录**、**我的**。栏位会避开手机底部安全区，在 iPhone 等设备上也能舒适点击。

### 任务

- **做任务赚积分**：点击任务横条即可加分，支持 13 项预设任务（洗手、吃饭、刷牙、按时睡觉、讲礼貌等）
- **横条列表**：任务与奖励以横条展示，名称过长时单行省略，便于日后多语言适配
- **按分值排列**：任务默认按积分从低到高展示
- **动效反馈**：加分时有飘字提示、星星徽章弹跳动画，还有撒花效果
- **震动反馈**：完成任务时可短振动提醒，可在「我的 → 设置」中开关，默认开启（部分 Android 手机支持，iPhone / iPad 不显示该选项）

### 奖励

- **换奖励花积分**：点击奖励横条即可兑换动画片、零食、玩具、游乐场等奖励
- **积分不足锁定**：当前星星不够时，奖励横条会灰显，点击会提示「积分不够哦」
- **兑换确认**：积分足够时会弹出确认框，避免误触
- **震动反馈**：兑换成功时可短振动提醒，同样可在设置中开关

### 记录

在底部栏点「记录」查看与编辑积分明细：

- **按日期查看**：默认显示今天的记录，顶部展示当日条数与净积分
- **全部记录**：一键切换查看所有历史，按日期分组展示
- **日历选择**：通过日历跳转到任意日期，有记录的日期带小圆点标记
- **回到今天**：浏览其他日期时可快速切回今天
- **编辑删除**：点「编辑」进入多选模式，选中一条或多条记录后删除；积分会相应调整，删除后无法恢复

### 我的

在底部栏点「我的」（显示宝贝头像）进入：

- **宝贝资料**：自定义昵称（最多 12 字）和头像（儿童 emoji + 十二生肖）
- **任务管理**：增删改任务、调整分值、更换图标，可启用或停用单项任务
- **奖励管理**：增删改奖励、调整分值、更换图标，可启用或停用单项奖励
- **设置**：震动反馈开关、修改密码、退出登录

顶部仍显示宝贝昵称、问候语和当前星星总数，方便随时查看积分。

### 登录与账号

- **家长登录**：打开应用需先登录家长账号
- **忘记密码**：输入邮箱获取验证码，在本页填写验证码并设置新密码；也可点击邮件中的重置链接，自动跳回并填入验证码
